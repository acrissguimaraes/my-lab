# My Lab: Data Science Fundamentals 🧪

Welcome! This repository is a collection of my studies, exercises, and projects developed during the **Data Science Fundamentals** course. Here, I document my journey from learning Python basics to performing exploratory data analysis (EDA) on real-world datasets.

## 📚 What I've Learned

### 1. Python Basics
- Variable types, lists, and dictionaries.
- Control flow: `if`, `for` loops, and `while` loops.
- Writing clean and functional code.

### 2. Data Manipulation (Pandas)
- Loading data from CSV and Excel files.
- Data cleaning: handling missing values (`isna`, `fillna`).
- Data transformation: creating new columns and filtering information.
- Grouping data for insights (Pivot tables and `groupby`).

### 3. Data Visualization (Matplotlib & Seaborn)
- Creating clear and professional charts.
- **Bar charts** for categorical analysis (e.g., Attrition by Department).
- **Line charts** for trends.
- Customizing charts: titles, labels, and the "Rotation Trick" for better readability.

---

## 🚀 Featured Project: HR Analytics (Employee Attrition)
In this project, I analyzed a dataset from IBM (HR Analytics) to understand why employees leave the company.

**Key Insights:**
- Calculated the turnover rate per department.
- Visualized which areas had the highest exit rates.
- Applied the **CRISP-DM** methodology to ensure the analysis was aligned with business goals.

> "Data alone doesn't make decisions; insights derived from data do."
