## 🧪 Quantitative & Qualitative Data Analysis

A collection of laboratory exercises focused on statistical analysis and data exploration using Python (Pandas, NumPy, Matplotlib, and Seaborn).

### 📈 Projects & Exercises Included:

1. **Laptop Pricing Exploratory Data Analysis** (`descriptive-stats-laptop-pricing.ipynb`)
   * **Objective:** Perform descriptive statistical analysis (Mean, Median, Mode, IQR) and explore data distribution for vehicle/laptop features and pricing.
   * **Key Visuals:** Frequency distribution histograms and correlation heatmaps.

2. **Statistical Sampling & Quantitative Analysis** (`quantitative-analysis-statistical-sampling.ipynb`)
   * **Objective:** Apply quantitative techniques to evaluate data samples, calculate proportions, and understand statistical significance.
